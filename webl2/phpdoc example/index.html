computeNeighborCandidates()

computeNeighborCandidates(integer  $neighbor) : array

A method to compute candidate names from a given neighbor.

The set of names are first extracted (local candidates), and for each of these names the score is updated. The score of a name is computed based on the number of times it was prefered.

Parameters

integer $neighbor

the neighbor

Returns

array —

The local candidates. key => value pairs, where key is a name id and value is the score for the name.
